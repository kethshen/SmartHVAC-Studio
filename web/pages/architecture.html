System Architecture

SmartHVAC Studio is an academic platform that integrates a web interface, cloud coordination, and offline HVAC simulation to support AI-assisted modeling and analysis of HVAC systems.

High-Level Architecture

  • Web Interface (SmartHVAC Studio)
    Used for experiment configuration, natural language input, and visualization of results.
  • Cloud Coordination (Firebase)
    Stores experiment metadata and manages file-based simulation outputs.
  • Simulation & Estimation Backend (Google Colab)
    Executes EnergyPlus simulations and EKF-based parameter estimation.
  • Experimental Validation
    Uses a single-zone HVAC laboratory setup for real-world validation.

Data Flow Overview

The web interface submits experiment requests, which are processed offline by the simulation backend. Results are uploaded to cloud storage and visualized through the web platform. 

User → Web Interface → Firebase → Colab (Simulation & EKF)
     ← Results & Plots ← Firebase Storage

Design Principles

  • Clear separation between interface and computation
  • Physics-based simulation with AI-assisted configuration
  • Modular and extensible research platform
  • Suitable for academic demonstration and evaluation

This platform is developed as part of an academic Final Year Project and is intended for research and educational purposes only.
